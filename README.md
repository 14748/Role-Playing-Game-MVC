## Utilizacion

####   Para su utilizacion simplemente clona el repositorio y mediante un interprete de php ejecuta la web. Sera necesaria una base mysql adecuada, se encuentra un scrip para generar esta en persistance

## Idea de arquitectura
![img](https://github.com/14748/Role-Playing-Game-MVC/assets/99936563/ab483b64-c4c0-48b1-860e-8754e4c3d217)

#### Se pretende hace una pagina con php usando arquitectura MVC.

#### Public view para ver las creaturas y detalles de estas ademas de poder inicar sesion y registrarse

#### Private view para ver las creaturas, sus detalles, modificarlas y borrarlas ademas de la posibilidad de anadir nuevas. Adicionalmente por logica en esta view se habilita la opcion de cerrar sesion.
